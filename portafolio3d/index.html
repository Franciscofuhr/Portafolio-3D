📜 FaltaUno

FaltaUno! is an app that seeks to facilitate the system that is used in sports complexes where they provide the service of renting courts, tournament organization and sporting events. It offers players the opportunity to find rivals or partners to play with, to book, cancel or modify the reservation and give them the possibility to make payments from the app. Also being able to visualize the information of the rivals individually or as a team and communicate with them in real time through a chat using the app

👩🏽‍🚀 Healthier

This project consisted in the development of a SPA (Single Page Application), which makes use of an API data, and then the information it's extracted and stored in a Data Base, so it can be used later at convenience.
Caracteristics of this project:
◽ Pagination showing 9 different recipes per page.
◽ Filter by HealthScore and by diets.
◽ Sorting (Alphabetical by recipe names (A-Z, Z-A).
◽ Detail charts for each recipe.
◽ Search bar.
◽ Controlled form to create and post recipes.
Technologies used for this project:
◾ Language: JavaScript, SQL.
◾ Data Base: PostgreSQL, Sequelize.
◾ Back-End: nodeJS, ExpressJS.
◾ Front-End: React, Redux, CSS in pure state.
